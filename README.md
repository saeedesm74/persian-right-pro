# 🇮🇷 Persian Right Pro

**افزونه‌ی راست‌به‌چپ‌ساز و فونت فارسی برای چت‌بات‌های هوش مصنوعی**

افزونه‌ای سبک و سریع برای مرورگرهای مبتنی بر Chromium (Chrome, Edge, Brave و...) که صفحه‌ی چت‌بات‌های هوش مصنوعی محبوب رو راست‌چین می‌کنه و فونت زیبای **وزیرمتن (Vazirmatn)** رو جایگزین فونت پیش‌فرض می‌کنه تا خوندن و نوشتن فارسی توی این ابزارها لذت‌بخش‌تر بشه.

## 📥 نصب و راه‌اندازی

از اونجایی که این افزونه هنوز در Chrome Web Store منتشر نشده، باید به‌صورت **Unpacked** نصبش کنی:

1. از بخش [Releases](../../releases) همین ریپازیتوری، آخرین نسخه‌ی فایل `persian-right-pro-vX.X.X.zip` رو دانلود کن.
2. فایل ZIP رو Extract (استخراج) کن.
3. توی مرورگر Chrome برو به آدرس:
   ```
   chrome://extensions
   ```
4. گزینه‌ی **Developer mode** (بالا سمت راست صفحه) رو روشن کن.
5. روی دکمه‌ی **Load unpacked** کلیک کن.
6. پوشه‌ای که Extract کردی (پوشه‌ی `persian-right-pro-vX.X.X`) رو انتخاب کن.
7. تمام! آیکون افزونه توی نوار ابزار مرورگر ظاهر می‌شه. وارد ChatGPT یا Claude یا هر سایت پشتیبانی‌شده‌ای بشو و تغییرات رو ببین 🎉

> این روش نصب برای مرورگرهای مبتنی بر Chromium مثل **Edge** و **Brave** هم عینا کار می‌کنه.

## 🛠 ساختار پروژه

```
persian-right-pro/
├── manifest.json        # تنظیمات و مجوزهای افزونه (Manifest V3)
├── content.js            # اسکریپت تزریق‌شده در صفحات چت‌بات‌ها
├── styles.css            # استایل‌های راست‌چین‌سازی
├── fonts.css             # تعریف فونت وزیرمتن
├── fonts/                # فایل‌های وزیرمتن (woff2)
├── popup.html/css/js     # رابط کاربری پاپ‌آپ افزونه
└── icons/                # آیکون‌های افزونه
```
